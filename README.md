## 📘 About This Application

Underwriting-Decision-Safety-Lab helps with loan approval decisions. It trains a base model to classify loan applications, then adjusts its confidence scores to improve reliability. The system uses clear rules to decide when to approve loans automatically and when to refer them for review. This way, it balances speed and care.

The app includes a dashboard you can open in your web browser. There, you will find loan reports, easy triage tools, and checks for your data quality. It uses simple visuals to present information, making it easy to understand.
